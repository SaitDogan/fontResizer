fontResize

jQuery plugin for font resize on JavaScript "resize" event.

Usage

  $(window).fontResizer({
    elements: [{elem: ".fontText", size: 1}]
  , baseWidth: 1280
  , startResize: true
  , sizeUnit: "em"
  });
